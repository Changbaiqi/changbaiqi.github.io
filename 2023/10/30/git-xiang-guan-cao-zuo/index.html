зҗҶи§Је·ҘдҪңеҢәе’ҢжҡӮеӯҳеҢәзҡ„еҢәеҲ«пјҹ

е·ҘдҪңеҢәпјҡе°ұжҳҜдҪ еңЁз”өи„‘дёҠзңӢеҲ°зҡ„зӣ®еҪ•пјҢжҜ”еҰӮзӣ®еҪ•дёӢtestgitйҮҢзҡ„ж–Үд»¶(.gitйҡҗи—Ҹзӣ®еҪ•зүҲжң¬еә“йҷӨеӨ–)гҖӮжҲ–иҖ…д»ҘеҗҺйңҖиҰҒеҶҚж–°е»әзҡ„зӣ®еҪ•ж–Үд»¶зӯүзӯүйғҪеұһдәҺе·ҘдҪңеҢәиҢғз•ҙгҖӮ
зүҲжң¬еә“(Repository)пјҡе·ҘдҪңеҢәжңүдёҖдёӘйҡҗи—Ҹзӣ®еҪ•.git,иҝҷдёӘдёҚеұһдәҺе·ҘдҪңеҢәпјҢиҝҷжҳҜзүҲжң¬еә“гҖӮе…¶дёӯзүҲжң¬еә“йҮҢйқўеӯҳдәҶеҫҲеӨҡдёңиҘҝпјҢе…¶дёӯжңҖйҮҚиҰҒзҡ„е°ұжҳҜstage(жҡӮеӯҳеҢә)пјҢиҝҳжңүGitдёәжҲ‘д»¬иҮӘеҠЁеҲӣе»әдәҶз¬¬дёҖдёӘеҲҶж”Ҝmaster,д»ҘеҸҠжҢҮеҗ‘masterзҡ„дёҖдёӘжҢҮй’ҲHEADгҖӮ

дҪҝз”ЁGitжҸҗдәӨж–Үд»¶еҲ°зүҲжң¬еә“жңүдёӨжӯҘпјҡ

з¬¬дёҖжӯҘпјҡжҳҜдҪҝз”Ё git add жҠҠж–Үд»¶ж·»еҠ иҝӣеҺ»пјҢе®һйҷ…дёҠе°ұжҳҜжҠҠж–Үд»¶ж·»еҠ еҲ°жҡӮеӯҳеҢәгҖӮ

з¬¬дәҢжӯҘпјҡдҪҝз”Ёgit commitжҸҗдәӨжӣҙж”№пјҢе®һйҷ…дёҠе°ұжҳҜжҠҠжҡӮеӯҳеҢәзҡ„жүҖжңүеҶ…е®№жҸҗдәӨеҲ°еҪ“еүҚеҲҶж”ҜдёҠгҖӮ

еёёз”ЁжҢҮд»Ө

и®ҫзҪ®gitз”ЁжҲ·еҗҚе’ҢйӮ®з®ұж ҮиҜҶ

жҹҘзңӢеҪ“еүҚgitз”ЁжҲ·дҝЎжҒҜ

еҲқе§ӢеҢ–д»“еә“

еўһеҠ ж–Үд»¶еҲ°жҡӮеӯҳеҢә

жҠҠж–Үд»¶жҸҗдәӨеҲ°жң¬ең°д»“еә“

жҹҘзңӢжҳҜеҗҰиҝҳжңүж–Үд»¶жңӘжҸҗдәӨ

жҹҘзңӢж–Үд»¶еҜ№жҜ”дёҠж¬ЎжҸҗдәӨдҝ®ж”№дәҶд»Җд№ҲеҶ…е®№

жҹҘзңӢcommitж—Ҙеҝ—

git logе‘Ҫд»ӨжҳҫзӨәд»ҺжңҖиҝ‘еҲ°жңҖиҝңзҡ„жҳҫзӨәж—Ҙеҝ—пјҢжҲ‘д»¬еҸҜд»ҘзңӢеҲ°жңҖиҝ‘дёүж¬ЎжҸҗдәӨпјҢжңҖиҝ‘зҡ„дёҖж¬ЎжҳҜ,еўһеҠ еҶ…е®№дёә333333.дёҠдёҖж¬ЎжҳҜж·»еҠ еҶ…е®№222222пјҢз¬¬дёҖж¬Ўй»ҳи®ӨжҳҜ 111111.еҰӮжһңе«ҢдёҠйқўжҳҫзӨәзҡ„дҝЎжҒҜеӨӘеӨҡзҡ„иҜқпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёе‘Ҫд»Ө git log вҖ“pretty=oneline жј”зӨәеҰӮдёӢпјҡ

commitзүҲжң¬еӣһйҖҖ

еҪ“з„¶пјҢеҰӮжһңеӣһйҖҖзҡ„зүҲжң¬еӨӘеӨҡзҡ„иҜқпјҢеҸҜд»Ҙиҝҷд№ҲеҶҷпјҡ

жҢүз…§зүҲжң¬еҸ·еӣһйҖҖ
