AJAX特点

优点

  1. 可以无需刷新页面而与服务器端进行通信。
  2. 允许你根据用户事件来更新部分页面内容。

缺点

  1. 没有浏览历史，不能回退
  2. 存在跨域问题（同源）
  3. SEO不友好（对搜索引擎不友好）

安装nodemon插件指令：

npm install -g nodemon

以后就可以用：nodemon *.js
